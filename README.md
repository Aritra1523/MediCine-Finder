# 💊 MediCine Finder

MediCine Finder is a web application that helps users search for medicines quickly and efficiently. It aims to reduce the effort of visiting multiple pharmacies by providing medicine information in one place.

---

## 📖 Overview

This project is built to simplify the process of finding medicines. Users can search for medicines and get relevant details through a clean and responsive interface. Pharmacists can manage their inventory efficiently by adding, editing, or deleting medicines.

---
## 🛠️ Tech Stack
### Frontend
- React.js   
- Tailwind CSS
- JavaScript  

### Backend
- Node.js
- mongoose
- Express.js

## 🚀 Features

### For Users
- User registration and login using a proper backend with **MongoDB (Mongoose)**  
- Search for medicines by name  
- View medicine details, including **price** and **stock availability**  
- Filter medicines by **price**  
- Contact the pharmacy directly via **call or WhatsApp**  
- Fast, responsive, and user-friendly interface  

### For Pharmacists
- Pharmacist registration and login using **MongoDB (Mongoose)**  
- Add new medicines with details such as **name, price, stock, and type** (e.g., tablets, strips, bottles, ml)  
- Edit or delete existing medicines  
- Manage medicine inventory efficiently   

---
## 🎯 Use Case

- Quickly search for medicines  
- See **availability and stock** of each medicine in nearby pharmacies  
- View **price** and **contact options** (call or WhatsApp) for the shop  
- Filter medicines by price or type  
- Save time by finding the required medicine without visiting multiple stores  
- Useful in **emergency situations**
